## Web アプリ (単語帳 & 進捗管理)

公開URL: https://keisukenishimura1991-sys.github.io/ENGLISH-Coarch/ （`main` への push で自動デプロイ）

ビルド不要の素のHTML/CSS/JSアプリなので、`index.html` をブラウザで開くだけでも使えます。

```bash
# ローカルで軽量サーバーを立てる場合 (任意)
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

### 機能
- **ダッシュボード**: 総単語数・復習待ち件数・マスター数・週間記事数・連続学習日数、直近7日間の復習グラフ
- **単語帳**: Leitner式の間隔反復（SRS）でのフラッシュカード復習 + 単語の追加・編集・削除
- **WSJ記事**: 記事タイトル・要約・メモを記録し、対応する Notion ページへのリンクを保存可能
- **コーチングテンプレート**: Kitamura-san とのセッション用テンプレートを保存・コピー

### データ保存
すべてブラウザの `localStorage` に保存されます（サーバー不要）。ヘッダーの「エクスポート」でJSONバックアップを取得、「インポート」で復元できます。

## Notion 連携（WSJ記事要約の正本）

WSJ記事の詳細な要約・重要単語表・注目フレーズは Notion の **英語 > ENGLISH NEWS** ページ配下に記事ごとのサブページとして記録しています。

記事本文をチャットで渡すと、以下の形式で高校生レベルの英語に要約し、Notion に登録します。
- `### Summary` — 要点の箇条書き要約
- `### 重要単語` — 単語と意味の対応表
- `### 注目フレーズ` — 実際に使える引用フレーズと解説

Web アプリの「WSJ記事」タブには、対応する Notion ページの URL を保存しておくことで、進捗管理と正本コンテンツを行き来できます。

## 運用方針：NotionとWebアプリは自動連携していません

- **正本（マスターデータ）は常に Notion**。毎朝8:00の自動復習もNotionのデータを直接参照します。
- Web アプリは「気分転換にフラッシュカードをめくりたい」「発音を聞きたい」ときに使う補助ツールという位置づけです。
- Web アプリに Notion の最新データを反映したい場合は、チャットでその時点のスナップショットを JSON で書き出してもらい、「インポート」ボタンで読み込んでください（都度手動）。
- 逆に Web アプリ側で追加・編集した単語は Notion には反映されません（片方向・非同期）。
